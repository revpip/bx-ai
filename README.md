# Business X-Ray AI

The AI strategy, prompt library, workflow definitions and evaluation standards for the Business X-Ray ecosystem.

## Purpose

This repository defines how AI is used across:

- Business X-Ray reports
- Assessment interpretation
- Recommendation drafting
- Follow-up emails
- Content Intelligence
- Website scanning summaries
- Client briefings
- Future AI assistants

## Core Principle

> AI assists judgement. Humans remain responsible for decisions.

## Rules

- Prompts should be versioned.
- AI output should be reviewed before client delivery.
- Recommendations must be explainable.
- Never pretend certainty where evidence is limited.
- Avoid storing sensitive client data in unnecessary places.
