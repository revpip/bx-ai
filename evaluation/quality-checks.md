# AI Quality Evaluation Checks

Every AI-generated client-facing output should be checked before delivery.

## Business X-Ray Report Checks

- Does it reflect the supplied business context?
- Does it avoid invented facts?
- Does it identify practical next actions?
- Does it avoid alarmist language?
- Does it explain uncertainty where needed?
- Does it avoid generic consultancy phrases?
- Is the recommendation linked to evidence?

## Content Intelligence Checks

- Does the content satisfy the likely search intent?
- Does it provide useful information beyond generic AI filler?
- Does it use British English where required?
- Does it avoid keyword stuffing?
- Does it include sensible internal linking opportunities?
- Does it avoid unsupported ranking promises?
- Does it include a clear commercial next step?

## Red Flags

- "In today's fast-paced digital landscape"
- "Unlock your potential"
- "Game-changer"
- Unsupported percentages
- Claims about guaranteed ranking
- Repetitive paragraph structures
- Advice outside the available evidence
